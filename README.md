Title
 # 🎟️ Campus Event Ticket Booking System
 This project is designed to solve real-world problems related to campus event management and online ticket booking using modern web technologies such as Node.js, Express.js, MongoDB, REST APIs, and React.js.
📌 Problem Description

Many universities and educational institutions still manage events manually through paper registrations, spreadsheets, social media posts, or Google Forms. This creates several issues such as:

Difficulty tracking attendees
Ticket duplication and overbooking
Lack of centralized event information
Time-consuming registration process
Poor communication between organizers and students
No secure user authentication

These problems negatively affect both students and event organizers.

🎯 Target Users
University Students
Campus Event Organizers
Student Clubs and Societies
University Administrators


❗ Why This Problem is Important

Campus events are important for student engagement, networking, entertainment, and academic activities. Without a proper system:

Students may miss important events
Organizers struggle to manage attendees
Event data becomes difficult to track
Manual processes waste time and resources

This system helps modernize and simplify event management within universities.

✅ Proposed Solution

The Campus Event Ticket Booking System provides a centralized platform where users can:

Browse upcoming campus events
Book tickets online
Manage event registrations
Track ticket availability in real time
Securely log into the system
Allow organizers to create and manage events efficiently

The system uses a RESTful API architecture with MongoDB database integration to handle real-world event booking operations effectively.
