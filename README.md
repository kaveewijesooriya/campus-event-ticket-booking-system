Title
 # 🎟️ Campus Event Ticket Booking System
 This project is designed to solve real-world problems related to campus event management and online ticket booking using modern web technologies such as Node.js, Express.js, MongoDB, REST APIs, and React.js.





📌 Problem Description

Many universities and educational institutions still manage events manually through paper registrations, spreadsheets, social media posts, or Google Forms. This creates several issues such as:

Difficulty tracking attendees
Ticket duplication and overbooking
Lack of centralized event information
Time-consuming registration process
Poor communication between organizers and students
No secure user authentication

These problems negatively affect both students and event organizers.
